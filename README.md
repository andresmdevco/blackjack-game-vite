# Blackjack Game Vite

Juego de Blackjack desarrollado con JavaScript utilizando Vite. 
Permite jugar contra la computadora con reglas básicas del juego.

## Demo
https://blackjack-game-andresmdevco.netlify.app/

## Tecnologías
- JavaScript
- Vite
- HTML
- CSS

## Ejecutar en local
Pasos para ejecutar proyecto:

1. Clonar repositorio
2. Ejecutar ```npm install``` para reconstruir los módulos de node
3. Correr el devServer ```npm run dev```
